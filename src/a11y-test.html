🧪 LembreMed - Suite de Testes de Acessibilidade

Testes automatizados com axe-core v4.10.2 + Auditoria de Contraste WCAG 2.1

📋 Sobre Esta Ferramenta INFO

Esta suite valida a conformidade do protótipo LembreMed com as diretrizes WCAG 2.1 AA. Importante: O teste axe-core requer que o protótipo esteja servido via servidor web (não funciona com arquivo file:// por restrições de segurança do browser).

Para testar o protótipo completo com axe-core, use: python -m http.server 8000 na pasta do projeto e acesse http://localhost:8000/a11y-test.html

🎨 Auditoria de Contraste de Cores Pronto

Validação matemática da luminância relativa entre pares de cores do design system. Testa WCAG 2.1 AA: 4.5:1 para texto normal, 3:1 para texto grande e componentes UI.

🔍 Scan axe-core Aguardando

Scan completo do DOM usando axe-core. Detecta violações de ARIA, contraste, labels, headings, landmarks e outras regras WCAG 2.1 AA automaticamente.

🧑‍🦯 Guia de Teste com Leitores de Tela

Testes automatizados cobrem ~57% das diretrizes WCAG. O restante requer teste manual com usuários reais ou leitores de tela.

🖥️ NVDA (Windows)

Grátis — nvaccess.org
Instale → abra o protótipo no browser → navegue com Tab
Dica: NVDA+N → Tools → Speech Viewer (ver texto falado)

🍎 VoiceOver (Mac/iOS)

Grátis — Built-in no sistema
Mac: Cmd+F5 para ligar/desligar
iPhone: Ajustes → Acessibilidade → VoiceOver
Dica: VO+U abre o Rotor para navegar por headings

🧩 axe DevTools (Browser)

Grátis — Extensão Chrome/Edge/Firefox
Instale da loja de extensões → clique "Scan"
Mostra violações com guia de correção passo a passo
